CURL a very hungry HTTP client

Command

curl -i https://api.github.com/test

curl: Stands for client URL. Its used to transfer data from and to a server using the command line or scripts.

-i: stands for --include. Shows hidden response headers of the HTTP reply.

https://api.github.com/test: The url where the GET request is issued and recieves the document that was asked for.
